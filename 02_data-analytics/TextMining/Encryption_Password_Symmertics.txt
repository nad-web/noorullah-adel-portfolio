
## Passwort-basierte symmetrische Verschlüsselung
wie man mithilfe von CyberChef eine Datei mit AES-128 verschlüsselt, wobei der Schlüssel aus einem Passwort abgeleitet wird.

Schritt 2: Schlüsselableitung mit PBKDF2
In CyberChef müssen Sie einen 128-bit Schlüssel aus dem gegebenen Passwort "saldkjf89374" ableiten:

Öffnen Sie CyberChef (https://gchq.github.io/CyberChef/)
Suchen Sie nach "Derive PBKDF2 key" und ziehen Sie diese Operation in den Rezeptbereich
Konfigurieren Sie die Operation wie folgt:
Passphrase: saldkjf89374
Iterations: 2000
Hash function: SHA512
Salt: 56
Key Size: 128 bits
Der generierte Schlüssel sollte nun angezeigt werden
Kopieren Sie diesen Schlüssel für den nächsten Schritt
Entfernen Sie die "Derive PBKDF2 key" Operation aus dem Rezept
Schritt 3: Verschlüsselung der Datei mit AES
Nachdem Sie den Schlüssel abgeleitet haben:

Ziehen Sie die "AES Encrypt" Operation in den Rezeptbereich
Fügen Sie den zuvor kopierten Schlüssel ein
Stellen Sie den Mode auf CBC (Cipher Block Chaining)
Geben Sie den spezifizierten IV ein: ......
Stellen Sie das Input-Format auf Hex und das Output-Format auf Raw ein
Laden Sie Ihre PDF-Datei in CyberChef hoch
Die Datei wird nun verschlüsselt
Laden Sie die verschlüsselte Datei herunter
